Objetivo

Monte corretamente a topologia da rede sem perder de vista o uso correto dos tipos de cabos em cada uma das conexões (straight-through, crossover, rollover e etc). Verifique também se as portas utilizadas (GigabitEthernet e FastEthernet) seguem as especificações da topologia, e se todas as conexões ficaram corretas (links verdes), até que todos os Switches e PCs estejam conectados. Após isso faça as configurações requeridas na parte de Tarefas.

Sumário de Comandos

Comando Modo e Propósito
line console 0 Comando global que muda o contexto para o modo de configuração de console.
vty 0 15 Comando global que muda o contexto para o modo de configuração vty para o escolo de linhas vty no comando.
login Modo de configuração de linha (console e vty). Diz ao IOS para pedir uma senha.
password valor-password Modo de configuração de linha (console e vty). Lista a senha necessária se o comando login estiver configurado.
interface digitar-o-número-da-porta Comando global que altera o contexto para o modo interaface - por exemplo, interface FastEthernet 0/1.
hostname nome Comando global que ajusta o nome de servidor do switch, que também é usado como a primeira parte do prompt de comando do switch.
exit Retorna ao modo anterior no modo de configuração.
end Encerra o modo de configuração e retorna ao modo ativo de qualquer um dos submodos de configuração.
Ctrl+Z Não é um comando, mas uma combinação de duas teclas que, se pressionadas juntas, têm a mesma função do comando end.

Tarefas do Lab

Tarefa 1: Monte a Topologia

  1. Insira os 03 Switches (modelo 2960) e os 07 computadores (qualquer dos modelos disponíveis) no simulador.

  2. Atente para os diferentes tipos de cabos requeridos (straight-through, crossover, rollover e etc). Faz alguma diferença cabos diretos ou cruzados na interligação dos switches nessa topologia? Pense sobre isso.

  3. Verifique se todos os links funcionaram corretamente, sinalizados com a luz verde.

  4. Em muitos casos é conveniente usar a ferramenta de notas do Cisco Packet Tracer, o que facilita a leitura da topologia, como nomear as interfaces que estão sendo utilizadas.

Tarefa 2: Configure os Switches

  1. Inicie pela configuração do Switch SW-01. O mais recomendado é que você só faça as demais configurações dos outros switches uma vez que tenha terminado o primeiro. Sempre que necessário utilize a ajuda no terminal de comandos CLI do equipamento, ou complete comandos com a tecla 'Tab'.

    1. Realize as configurações através do terminal de console do PC Z.

    2. A partir do modo de Configuração Global, configure o hostname do equipamento como 'SW-01'.

    3. Coloque uma senha 'admin' para entrar no modo enable do equipamento.

    4. Coloque uma senha de acesso local (console) 'iflocal' no equipamento.

    5. Coloque uma senha para acesso remoto (linhas vty) pelo protocolo Telnet 'ifremoto' no equipamento.

    6. Através do comando show running-config, confira se as configurações aplicadas estão corretas.

    7. Salve as configurações do equipamento (startup-config) ao final.

  2. Repita os procedimentos acima, alterando o respectivo hostname, para o Switch SW-02.

  3. Repita os procedimentos acima, alterando o respectivo hostname, para o Switch SW-03.
